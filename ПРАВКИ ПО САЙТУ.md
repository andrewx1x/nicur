Сейчас на сайте почти везде только текст + публикации. Чтобы сайт выглядел цельно и интересно, нужно добавить новые типы секций в `PageSections.astro` и `global.css`:

### 1. Секция `stats` — ключевые цифры
Крупные числа в карточках с подписью. Визуально:

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│  71 856  │  │    42    │  │     6    │
│ компаний │  │  города  │  │GIS-слоёв │
└──────────┘  └──────────┘  └──────────┘
```

JSON-формат:
```json
{
  "type": "stats",
  "title": "Ключевые цифры",
  "items": [
    { "value": "71 856", "label": "компаний оценено в Енисейской Сибири" },
    { "value": "42", "label": "города в индексе качества жизни" },
    { "value": "6", "label": "слоёв GIS-платформы" }
  ]
}
```

### 2. Секция `timeline` — этапы / история
Вертикальная линия с точками и описаниями. Для истории строительства, этапов проекта и т.п.

```json
{
  "type": "timeline",
  "title": "Этапы развития",
  "items": [
    { "year": "2023", "title": "TRL 1", "text": "Описание этапа" }
  ]
}
```

### 3. Секция `features` — карточки с иконкой и описанием
Для компонентов платформы, направлений деятельности, услуг компании.

```json
{
  "type": "features",
  "title": "Направления",
  "items": [
    { "icon": "🏭", "title": "Заголовок", "text": "Описание" }
  ]
}
```

### 4. Секция `table` — таблица данных
Для сравнительных данных (сорбенты, цены на уголь, рейтинги).

```json
{
  "type": "table",
  "title": "Заголовок таблицы",
  "headers": ["Колонка 1", "Колонка 2"],
  "rows": [
    ["Значение 1", "Значение 2"]
  ]
}
```
